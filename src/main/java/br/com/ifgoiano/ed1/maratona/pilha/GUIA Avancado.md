## 🏆 Lista de Exercícios de Pilhas em Java — **Avançado**

### 📘 **Nível 1: Introdução a Problemas Competitivos (49–58)**
🔹 *Objetivo: resolver problemas de juízes online (Beecrowd, Codeforces, LeetCode) que usam pilhas.*

49. **Beecrowd 1110 — Throwing Cards Away.**  
    Simular descarte de cartas em sequência circular.

50. **Beecrowd 1062 — Railroads.**  
    Verificar se é possível reorganizar vagões usando pilha.

51. **Codeforces 5C — Longest Regular Bracket Sequence.**  
    Encontrar a maior subsequência balanceada de parênteses.

52. **Beecrowd 1025 — Where is the Marble?.**  
    Aplicação de busca em pilha ordenada.

53. **LeetCode 20 — Valid Parentheses.**  
    Clássico de validação de brackets.

54. **LeetCode 155 — Min Stack.**  
    Implementar pilha que retorna o mínimo em `O(1)`.

55. **LeetCode 232 — Implement Queue using Stacks.**  
    Simular fila usando duas pilhas.

56. **LeetCode 71 — Simplify Path.**  
    Processar caminhos Unix com `/`, `.`, `..` usando pilha.

57. **Codeforces 612C — Replace To Make Regular Bracket Sequence.**  
    Contar trocas mínimas para balancear expressão.

58. **HackerRank — Maximum Element.**  
    Manter controle do maior elemento em pilha dinâmica.

### 📗 **Nível 2: Pilha Monotônica (59–70)**
🔹 *Objetivo: dominar a técnica de pilha monotônica para otimização de algoritmos.*

59. **Problema: Next Greater Element (Próximo Maior à Direita).**  
    **Exemplo:**
    ```txt
    [4, 5, 2, 10] → [5, 10, 10, -1]
    ```

60. **Problema: Next Greater Element à Esquerda.**

61. **Problema: Next Smaller Element (Próximo Menor).**

62. **LeetCode 739 — Daily Temperatures.**  
    Quantos dias até temperatura mais quente.

63. **LeetCode 496 — Next Greater Element I.**

64. **LeetCode 503 — Next Greater Element II (array circular).**

65. **LeetCode 42 — Trapping Rain Water.**  
    Calcular água acumulada entre barras usando pilha.

66. **LeetCode 84 — Largest Rectangle in Histogram.**  
    Maior retângulo em histograma com pilha monotônica.

67. **LeetCode 85 — Maximal Rectangle.**  
    Extensão 2D do problema do histograma.

68. **Codeforces 5E — Bindian Signalizing.**  
    Torres que enxergam umas às outras.

69. **LeetCode 901 — Online Stock Span.**  
    Calcular span de preços de ações em tempo real.

70. **LeetCode 456 — 132 Pattern.**  
    Detectar padrão numérico usando pilha monotônica.

### 📙 **Nível 3: Problemas Avançados de Maratona (71–80)**
🔹 *Objetivo: resolver problemas complexos que combinam pilhas com outras técnicas.*

71. **SPOJ ANARC09A — Seinfeld.**  
    Mínimo de inversões para balancear chaves.

72. **SPOJ HISTOGRA — Largest Rectangle in a Histogram.**  
    Versão SPOJ do problema clássico.

73. **Codeforces 5C — Longest Regular Bracket Sequence (Otimizado).**

74. **LeetCode 394 — Decode String.**  
    Decodificar strings com padrão `k[encoded_string]`.

75. **LeetCode 636 — Exclusive Time of Functions.**  
    Calcular tempo de execução de funções aninhadas.

76. **LeetCode 726 — Number of Atoms.**  
    Parsear fórmulas químicas usando pilha.

77. **LeetCode 895 — Maximum Frequency Stack.**  
    Pilha que retorna elemento mais frequente.

78. **Codeforces 5E — Bindian Signalizing (Solução Otimizada).**

79. **AtCoder ABC 334E — Christmas Color Grid.**  
    Problema de coloração com DFS iterativo.

80. **ICPC World Finals 2019 — Traffic Blights.**  
    Simulação complexa de semáforos com pilhas.
