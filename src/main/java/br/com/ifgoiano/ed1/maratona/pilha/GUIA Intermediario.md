## 🔰 Lista de Exercícios de Pilhas em Java — **Intermediário**

### 📗 **Nível 2: Notação Polonesa e Avaliação de Expressões (31–40)**
🔹 *Objetivo: converter e avaliar expressões aritméticas usando pilhas.*

31. **Converter uma expressão infixa simples para pós-fixa (polonesa reversa).**
    **Exemplo:**
    ```txt
    "a+b" → "ab+"
    ```

32. **Avaliar uma expressão pós-fixa com números inteiros.**
    **Exemplo:**
    ```txt
    "5 3 + 2 *" → 16
    ```

33. **Converter expressão infixa para pós-fixa com operadores `+`, `-`, `*`, `/`.**

34. **Avaliar expressão pós-fixa com números decimais.**

35. **Converter expressão pós-fixa de volta para infixa.**

36. **Implementar calculadora que aceita expressão infixa e retorna resultado.**

37. **Converter expressão infixa para pré-fixa (notação polonesa).**
    **Exemplo:**
    ```txt
    "a+b" → "+ab"
    ```

38. **Avaliar expressão pré-fixa.**
    **Exemplo:**
    ```txt
    "+ 5 3" → 8
    ```

39. **Lidar com operadores de precedência e parênteses na conversão infixa-pós-fixa.**

40. **Criar uma calculadora científica simples (suporta `^`, `sqrt`, `%`).**

---

### 📙 **Nível 3: Recursão e Histórico (41–48)**
🔹 *Objetivo: simular recursão, navegação e controle de estados com pilhas.*

41. **Simular a pilha de chamadas de uma função recursiva simples (fatorial).**

42. **Converter função recursiva de Fibonacci para versão iterativa com pilha.**

43. **Implementar navegador com histórico (`Back/Forward`) usando duas pilhas.**

44. **Simular a execução de comandos `Undo/Redo` em um editor.**

45. **Resolver o problema das Torres de Hanoi usando pilhas.**

46. **Percorrer uma árvore binária em pré-ordem sem recursão (usando pilha).**

47. **Percorrer uma árvore binária em pós-ordem sem recursão.**

48. **Implementar DFS (Busca em Profundidade) em grafo usando pilha explícita.**
